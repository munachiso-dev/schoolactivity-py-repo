# School Activity Calculator

This is a simple Python calculator program created as a school activity. The program asks the user to input two numbers and then performs basic arithmetic operations such as:
- Addition
- Subtraction
- Multiplication
- Division

## How It Works
1. The user is prompted to enter the first number.
2. The user is prompted to enter the second number.
3. The user selects an operation.
4. The prgram outputs the result of the chosen operation.
